# script[defer]

```html
<head>
  <script src="slow.js"></script>
  <!-- We can't even start parsing this HTML until slow.js has been loaded and executed! -->
```

# a[rel=noopener]

```html
<a href="https://slow-page.com" target="_blank">Link</a>
```

This link sets `window.opener` in the opened page to the current window.

This forces them onto the same thread!
