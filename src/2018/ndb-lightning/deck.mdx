# ndb
## node debugging for the modern dev

---

## ndb

- Use familiar Chrome Developer Tools
- Easily debug your tests
- Re-run npm scripts from within the debugger
- No need to find magic node args, `ndb` does the work for you
- Let's check it out

---

## install it

<pre style={{fontSize: '60px'}}>npm install -g ndb</pre>

---

## use it

<pre style={{fontSize: '60px', textDecoration: 'line-through'}}>npm test</pre>

<pre style={{fontSize: '60px'}}>ndb npm test</pre>
